/** Объект Товар в глабльном представлении должен быть такого типа: goodName = { prop1: , ... propN: , } как понимаю, тогда его имя относится к целому классу подобных товаров. А индивидуализация достигается за счет подборки его свойств. К примеру объекты: const apple = { type: 'product', sort: 'antonovka', price: 34, value: 10, }; const pear = { type: 'product', sort: 'mramornaya', price: 40, value: 13, }; Любое яблоко относится к группе объектов типа apple. а груша к pear. Индивидуализировать можем за счет Сорта, Цены и тд. Так же эти объекты относятся к одной группе товаров, позволит их вывести в любом месте сайта по свойству type: 'product'. */
